Display process of a static page.

  1. The server receives the request from the client.
  2. It goes into view.py file, then it finds out that it is a GET request from the client
  3. It chekcs the route, then it sends back to the client the right HTML file which is in the templates
  4. Then the client will get the static web page, and be able to see it on their navigator!
