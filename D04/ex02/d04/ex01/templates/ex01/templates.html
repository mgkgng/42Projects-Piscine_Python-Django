Template engine.

  • Blocks

    • use {% block content_name %} ...contentblock... {% endblock %} syntax in order to use blocks!


  • Loops for ... in

    • use {% for element in iterable %} ...codeblock... {% endfor %} syntax in order to use for loops!


  • if control structures

    • use {% if condition %} ... {% endif %} syntax in order to if control structures!


  • The display of the passing context variables

    • use dictionnary {'key' : 'value', ...} as the third parameter of render function as you want to get the response from the server!
